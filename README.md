# ループアニメーションメーカー

![スクリーンショット](images/ss-v2.jpg)

## 機能概要

MP4動画から自然にループしそうな最適区間を自動で探して、高速でシームレスループアニメを作成できるTkinter+OpenCV製GUIツールです。

### 主な機能

- 動画全体から「SSIM（またはエッジ比較）スコアが最高の自然なループ区間」を自動で検索・抽出
- 最小ループ長・最大ループ長・検索間隔を自由に設定可能
- 事前に全フレームを160×90に縮小してメモリに保持 → 劇的な高速化
- フェード合成（任意）とエッジ（輪郭）比較モードも搭載
- 最適区間＋オプションでフェードを付けたループ動画を自動出力

### 一言で言うと

「自動最適ループアニメメーカー」

## 使い方

1. **アプリを起動する**

    ターミナルで`python loop-animation-maker-v2.py`を実行（事前に`pip install tkinterdnd2 opencv-python pillow scikit-image numpy`を済ませておけよ）。

2. **動画を読み込む**

    「動画を読み込む」ボタンか、ウィンドウにMP4ファイルをドラッグ＆ドロップ。

3. **検索パラメータを設定する**

    - 最小ループ長（デフォルト60フレーム）
    - 最大ループ長（デフォルト300フレーム）
    - 検索間隔（デフォルト20フレーム）
    - 必要なら「フェード合成」「エッジ比較」にチェックを入れる

4. **最適ループを自動検索して出力**

    「最適ループを自動検索して出力」ボタンをクリック。数秒〜十数秒で検索が完了し、`bestloop-YYYYMMDDHHMMSS.mp4`として保存される。

5. **完了を確認**

    ステータスバーに最適区間情報（開始フレーム→終了フレーム、SSIMスコア）と保存先が表示される。

## 必要環境

- Python 3.10以上
- 必要なライブラリはソースコードの先頭に書いてあります。
